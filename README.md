# ⚡ Code Sync - Real-Time Collaborative Editor

A highly scalable, real-time collaborative code editor built with React, Node.js, and Socket.io. Features a **secure, isolated remote code execution engine** powered by Docker, allowing users to compile and run C++, Python, Java, and JavaScript instantly.

## ✨ Core Features
- **👥 Real-Time Collaboration**: Type, edit, and code together with your peers with zero latency using WebSockets.
- **🐳 Secure Code Execution**: Built a custom remote code execution engine using Docker containers to safely compile and run code in multiple languages (C++, Python, Java, JS).
- **🎨 Premium UI/UX**: Designed a Cyberpunk-themed, glassmorphism-inspired user interface with Framer Motion animations and custom neon accents.
- **💬 In-Room Chat**: Integrated real-time team chat with unread message notifications and sound alerts.
- **🛠️ VS Code-like Experience**: Integrated Monaco Editor with syntax highlighting, auto-formatting, and intelligent code completion.

## 💻 Tech Stack
- **Frontend**: React.js, Tailwind CSS, Framer Motion, Monaco Editor, React-Hot-Toast
- **Backend**: Node.js, Express.js
- **Real-Time Communication**: Socket.io
- **Infrastructure**: Docker (for isolated code compilation environments)
